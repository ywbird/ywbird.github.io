## 정규 표현식 검색하는 법

```python
>>> import re
>>> p = re.compile('《표현식》')
>>> m = p.match('《검사할 문자》')
>>> print(m)
<re.Match object; span=(0, 6), match='《결과로 나온 문자》'>
```

이건 기초적인 `match()`함수를 이용한확인 법이다.

## 정규 표현식 검색

### 점(.)

점은 모든 글자입니다.
다음과 같이 표현식을 구축했습니다.

```
nya.n
```

위의 표현식은 다음과 같이 해석할 수 있습니다.

> `nay` `《아무 글자》` `n`
