## 간단한 목차

- 숫자 변수
- 문자열 변수
- 리스트 변수
- 참, 거짓 변수

## 숫자 변수

```lua
> a = 1
> a
1
```

걍 숫자입니다.

```lua
> a = 1
> b = 2
> a + b
3
```

이렇게 변수끼리 더할 수 있습니다.

## 문자열 변수

```lua
> a = 'banana'
> a
banana
```

문자열이다.

```lua
> a = 'lua '
> b = 'is '
> c = 'easy'
> a + b + c
lua is easy
```

문자열끼리 더할 수 있습니다.
